# How to use ?
open "question3.md" and run script. Then you should enter circiut components in below format.

- R node1 node2 r

- C node1 node2 c

- L node1 node2 l

- VCC k k’ j j’ gm

- VCVS k k’ j j’ A

- VS i j E

- CS i j C

- OP node+  node-  outNode gain
